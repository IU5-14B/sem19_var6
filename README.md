# Решение задачи линейного программирования

## Описание проекта
Программа для решения задачи линейного программирования методом симплекса.

## Постановка задачи
Максимизировать: 0.10x + 0.07y + 0.03z

При условиях:
1. x + y + z = 100
2. x ≤ (1/3)y
3. z ≥ 0.25(x + y)
4. x ≥ 0, y ≥ 0, z ≥ 0

## Сборка и запуск

### С использованием CMake:
```bash
mkdir build
cd build
cmake ..
make
./lp_solver
